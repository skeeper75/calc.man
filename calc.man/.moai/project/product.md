# calc.man - 웹 기반 계산기 애플리케이션

## 프로젝트 개요

calc.man은 웹 브라우저에서 동작하는 계산기 애플리케이션이다. React와 shadcn/ui를 기반으로 직관적이고 반응형인 UI를 제공하며, 기본적인 산술 연산부터 고급 계산 기능까지 지원하는 것을 목표로 한다.

### 미션

누구나 쉽고 빠르게 사용할 수 있는 웹 계산기를 제공하여, 별도 앱 설치 없이 브라우저만으로 다양한 계산 작업을 수행할 수 있도록 한다.

### 비전

가볍고 빠른 웹 계산기를 통해 일상적인 계산 작업의 접근성을 높이고, 확장 가능한 구조로 향후 공학/과학 계산 기능까지 지원하는 플랫폼으로 성장한다.

---

## 대상 사용자

### 주요 사용자

- **일반 사용자**: 일상적인 사칙연산, 퍼센트 계산, 단위 변환 등 기본 계산이 필요한 사람
- **학생**: 수학 과제나 학습 중 빠른 계산이 필요한 학생
- **사무직 종사자**: 업무 중 간단한 수치 계산이 필요한 직장인

### 부가 사용자

- **개발자**: 진법 변환, 비트 연산 등 프로그래밍 관련 계산이 필요한 개발자
- **엔지니어**: 공학적 수식 계산이 필요한 기술 종사자

---

## 핵심 기능

### 기본 계산 기능

1. **사칙연산**: 덧셈, 뺄셈, 곱셈, 나눗셈
2. **소수점 처리**: 소수점 입력 및 정밀 계산
3. **부호 전환**: 양수/음수 전환
4. **퍼센트 계산**: 백분율 변환 및 퍼센트 기반 계산
5. **초기화 기능**: 전체 초기화(AC) 및 마지막 입력 삭제(C)

### 사용자 편의 기능

6. **계산 이력**: 이전 계산 결과 목록 조회 (SQLite 저장)
7. **키보드 입력 지원**: 숫자 키패드 및 연산자 키보드 단축키
8. **복사/붙여넣기**: 계산 결과 클립보드 복사
9. **반응형 디자인**: 데스크톱/태블릿/모바일 화면 크기 대응

### 확장 기능 (향후 고려)

10. **공학 계산기 모드**: 삼각함수, 로그, 거듭제곱 등
11. **진법 변환**: 2진수, 8진수, 16진수 변환
12. **테마 전환**: 라이트/다크 모드 지원

## 사용 시나리오

### 시나리오 1: 일상 계산

사용자가 브라우저에서 calc.man에 접속하여 장보기 예산을 계산한다. 여러 항목의 가격을 더하고, 할인율(퍼센트)을 적용하여 최종 금액을 확인한다.

### 시나리오 2: 연속 계산

사무직 종사자가 여러 건의 견적을 연속으로 계산한다. 각 계산 결과가 이력에 저장되어, 이전 결과를 참조하거나 재사용할 수 있다.

### 시나리오 3: 모바일 접근

이동 중인 사용자가 스마트폰 브라우저에서 calc.man에 접속하여 빠르게 계산을 수행한다. 반응형 UI로 모바일 화면에서도 편리하게 버튼을 누를 수 있다.

### 시나리오 4: 키보드 기반 빠른 입력

개발자가 데스크톱에서 키보드만으로 빠르게 수식을 입력하고 결과를 확인한 뒤, 결과를 클립보드에 복사하여 코드에 붙여넣는다.

## 프로젝트 목표

### 단기 목표 (1-2개월)

- 사칙연산 및 기본 계산 기능 구현
- 반응형 UI 구축 (shadcn/ui + Tailwind CSS)

- 공학 계산기 모드 추가

- PWA 지원으로 오프라인 사용 가능
- 계산 이력 검색 및 즐겨찾기

### 장기 목표 (6개월 이후)

- 진법 변환 및 비트 연산 기능
- 단위 변환기 통합
- 수식 시각화 기능
- 다국어 지원
